会話の待ち時間、表情、日記の記憶、設定まわりをまとめて改善したアップデートです。

## 変わったこと

### 会話が早く、途切れにくく

- Gemini / Claude APIの回答を受信しながら表示し、最初の一文ができた段階からVOICEVOXで話し始めます
- 返答待ちに短い相づちと考え中の音を入れ、回答が始まったら自然に切り替えます
- 選択中の会話AIが使えない場合は、利用できる別のAIへ自動で切り替えます
- 同じ言い回しや直近の話題を繰り返しにくくしました

### 表情と動き

- 聞く・考える・話す・待機の状態に合わせて、表情と動きがなめらかに変わります
- 困り顔、悲しい顔、考え中などを会話内容に合わせて使います
- 自動移動を調整し、画面左端で止まることがある問題を修正しました

### 日記が時間につながるように

- 朝は昨日、月曜は先週の日記から、実在する一行だけを自然な頻度で振り返ります
- 新しく作った日記に時刻の根拠がある場合だけ、翌日の同じ時間帯に思い出します
- 保存済み日記を会話AIへ送る設定は初期値OFFです。Mac内だけの振り返りはOFFのまま使えます
- 日記を編集・削除した場合、古い内容や時刻対応を次の会話へ持ち越しません

### 使い勝手とデータ管理

- 呼び名、好み、ことば、思い出、日記を確認・編集・個別削除できるデータ管理画面を追加しました
- キャラクターへ触れると15 / 25 / 90分のクイックポモドーロが現れます
- ミュートボタンへ触れると、10〜100%の音量スライダーが現れます
- ニュースや生活ハックの出典を「☆ 保存」し、あとから「気になる記事」で見返せます
- BGMのおすすめをYouTubeで検索でき、吹き出し本文を部分コピーできます

### 安全性・プライバシー

- Claude APIキーをmacOSの安全な保存機能（Keychain連動）で暗号化して保存します
- びくたんが見られない画面や部屋の様子を、見たように話すことを抑えました
- 記憶・履歴・APIキーの保存状況をデータ管理画面で確認できます

## インストール

1. `bikutan-mac-universal-v0.2.0.zip` をダウンロードして解凍
2. `びくたん.app` を「アプリケーション」フォルダへ移動（上書きでOK。設定・育成データは引き継がれます）
3. 公証未対応のため、初回は「システム設定」→「プライバシーとセキュリティ」→「このまま開く」を選んでください

「壊れているため開けません」と表示される場合に限り、Release記載のSHA-256を確認してから次を実行してください:

```bash
xattr -cr /Applications/びくたん.app
```

ダウンロードしたZIPのSHA-256:

```text
2e89be79fab3bd861cbd202ab0efebdc3a838a7aa0cfbabe118242a201d7a606
```
