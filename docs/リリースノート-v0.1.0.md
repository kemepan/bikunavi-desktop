Live2Dキャラクター「びくにたん」がmacOSのデスクトップに常駐するAIマスコットアプリです。

## できること

- 透明背景・最前面のLive2Dデスクトップマスコット（物理演算つき）
- AIとの会話（Codex CLI / Claude Code CLI / Gemini CLI / Claude APIキーのいずれかで。無くても他機能は動作）
- VOICEVOX「猫使ビィ」による読み上げと口パク
- ニュース・生活ハックを混ぜた自動セリフ、日替わり気分占い、ポモドーロタイマー
- ことば帳・思い出帳・質問への回答でキャラが育つ
- マイクからのローカル音声入力（whisper.cpp、音声は外部送信しません）

## 動作環境

- macOS（Apple Silicon / Intel 両対応のユニバーサルビルド）
  - Intel Mac での動作確認は Rosetta 検証のみです。問題があれば Issue で教えてください
- 読み上げには [VOICEVOX](https://voicevox.hiroshiba.jp/)（無料）が必要です

## インストール

1. `bikutan-mac-universal-v0.1.0.zip` をダウンロードして解凍
2. `びくたん.app` を「アプリケーション」フォルダへ
3. 公証未対応のため初回のみターミナルで以下を実行してから開いてください:

```
xattr -cr /Applications/びくたん.app
```
