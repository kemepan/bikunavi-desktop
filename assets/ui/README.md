# UIアイコン置き場

ここに以下のファイルを置くと、キャラ上のミュートボタンが絵文字（🔊/🔇）から画像に切り替わります。片方だけ置いた場合はその状態のみ画像になり、残りは絵文字のまま動きます。

| ファイル名 | 状態 | イメージ |
|---|---|---|
| `sound-on.png` | 音が出る状態 | 笑顔のびくたん（2026-07-19 適用済み） |
| `sound-off.png` | ミュート中 | バッテン（×）マーク付きマスクをつけたびくたん（2026-07-19 適用済み） |

## 仕様

- PNG・背景透過
- 正方形（推奨 128×128px。表示は約22px なので細部は潰れます。輪郭はっきりめが吉）
- 差し替え後は `./scripts/deploy-launchagent.sh` で反映
