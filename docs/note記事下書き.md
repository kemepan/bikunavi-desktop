# 自作Live2Dキャラを、Macのデスクトップに住まわせた話（無料配布します）

【スクショ: デスクトップの隅にいるびくにたん。吹き出しでひとこと喋っている場面】

こんにちは、Live2D・3Dモデラーのびくに（〔@SNSアカウント〕）です。

ポートフォリオサイト（[bikunitan.online](https://bikunitan.online)）で動かしているLive2Dキャラクター「びくにたん」を、**Macのデスクトップに常駐するAIマスコット**にしてみました。せっかくなので、無料で配布します。

## びくにたんは何をしてくれるの？

- **デスクトップの隅に住みます。** 透明背景・最前面で、作業の邪魔にならないサイズ。ドラッグで好きな場所へ動かせます。Live2Dの物理演算つきなので、つつくとちゃんと揺れます。
- **話しかけると答えます。** `⌘⇧B` でどこからでも会話欄が開きます。マイクの音声入力にも対応（ローカル処理なので音声は外部に送られません）。
- **声を出して喋ります。** VOICEVOXの「猫使ビィ」の声で読み上げます。
- **勝手にひとりごとを言います。** AIニュースや生活ハックの見出しを拾って、30秒〜2分おきに軽い雑談をします（オフにもできます）。
- **育ちます。** ことば帳・思い出帳・好みの質問を通じて、あなた用のびくたんになっていきます。質問は選択肢ボタンでサクッと答えられます。
- **ちょっとした道具にもなります。** ポモドーロタイマー、日替わりの気分占い、今日の要点を残す日記。

## 動作環境と必要なもの

- **macOS**（Apple Silicon / Intel 両対応のユニバーサル版です。Intel Macでの動作確認は限定的なので、動かなかったら教えてください）
- **[VOICEVOX](https://voicevox.hiroshiba.jp/)**（無料）— 読み上げに使います。入れなくても動きますが、声が出ません。
- **会話AI（任意）** — AIとの自由会話を使う場合だけ、次のいずれかが必要です:
  - Codex CLI / Claude Code CLI / Gemini CLI のどれか（ログインして使っている人はそのまま使えます）
  - または Claude APIキー（メニューから設定）
  - **何もなくても、自動セリフ・占い・ポモドーロ・育成は動きます。**

## ダウンロードとインストール

1. 〔GitHub ReleasesのURL〕から `bikutan-mac-universal-v0.1.0.zip` をダウンロード
2. 解凍して `びくたん.app` を「アプリケーション」フォルダへ移動
3. **初回起動は下の手順が必要です（重要）**

### 初回起動の手順（Gatekeeper対策）

このアプリは個人配布のため、Appleの公証（ノータリゼーション）を通していません。そのままダブルクリックすると **「壊れているため開けません」** と表示されます。壊れていません。安心してください。

ターミナル（アプリケーション → ユーティリティ → ターミナル）を開いて、次の1行を貼り付けてEnterを押してください:

```
xattr -cr /Applications/びくたん.app
```

そのあとダブルクリックすれば起動します。この操作は初回だけです。

## 会話AIの設定

メニューバーの🌱アイコン →「会話AI」から選べます。既定の「自動」は、Mac内で見つかったAI（Codex → Claude Code → Gemini → Claude API の順）を使います。
