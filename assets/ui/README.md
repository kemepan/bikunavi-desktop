# UIアイコン置き場

ここに以下の2ファイルを置くと、キャラ上のミュートボタンが絵文字（🔊/🔇）から画像に切り替わります。無い場合は絵文字のまま動きます。

| ファイル名 | 状態 | イメージ |
|---|---|---|
| `sound-on.png` | 音が出る状態 | 通常のびくたん顔 |
| `sound-off.png` | ミュート中 | バッテン（×）マーク付きマスクをつけたびくたん |

## 仕様

- PNG・背景透過
- 正方形（推奨 128×128px。表示は約22px なので細部は潰れます。輪郭はっきりめが吉）
- 差し替え後は `./scripts/deploy-launchagent.sh` で反映
