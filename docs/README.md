運用メモ:

- README は入口として短く保ち、細かい判断や作業手順は `docs/` に寄せる。
- 作業中に見つけた未解決の違和感は、まず [`ROADMAP.md`](ROADMAP.md) に追記する。
- 実装の履歴やなぜそうしたかは [`../DEVELOPMENT_NOTES.md`](../DEVELOPMENT_NOTES.md) に残す。
- 外部配布・公開に関わる話は [`RIGHTS_CHECK.md`](RIGHTS_CHECK.md) と [`公開手順-v0.1.0.md`](公開手順-v0.1.0.md) を優先して更新する。
